# cape-cash

## Run Locally

**Prerequisites:** [Node.js](https://formulae.brew.sh/formula/nvm)

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env](.env) to your Gemini API key
3. Set the `GSA_API_KEY` in [.env](.env) to your [GSA API key](https://open.gsa.gov/api/perdiem/#user-requirements)
4. Run the app:
   `npm run dev`
